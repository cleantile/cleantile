Class: MetaFile

Defined in: bin/metafiles.coffee

Overview

Describes a single file that can be directly copied into a distribution.

Property Summary

(String) name
this file's name

Instance Method Summary

  • # (Promise) copy(opts) Copy this metafile into a distribution directory.

Constructor Details

# (void) constructor(name, opts = {})

name this file's name

Parameters:

  • opts ( Object ) — options to configure the file copy.

Options Hash: (opts):

  • fileDir ( String ) — the path to the file's directory relative to the repository root, defaulting to "".
  • fileOutDir ( String ) — the path to the file's output directory, relative to the build directory. Defaults to matching fileDir.
