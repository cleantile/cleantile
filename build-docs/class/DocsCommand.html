Class: DocsCommand

Defined in: bin/_docs.coffee

Overview

Compiles documentation, and optionally publishes it to gh-pages.

Property Summary

(Object) opts
contains CLI arguments and internal options.
(Object) deps
additional dependencies for the documentation pages.

Instance Method Summary

  • # (Promise) buildDocs() Builds documentation pages for local use, or use on gh-pages.
  • # (Promise) compileDemos(elements) Find and compile both global demos as well as per-element demos.
